===================================================================================================
Дана флешка призначена для відкривання і редагування .planner файлів. 
Працює в двох режимах: десктоп і веб. Відмінності цьих двох режимах полягає у способі загрузки файла у флешку.
У десктоп режимі підгрузка файла робиться за допомогою однієї кнопки "open", а збереження за допомогою кнопки "save".
У веб режимі підгрузка файла(та збереження) виконується за допомогою JavaScript-а.
===================================================================================================
Десктоп версій:
1) відкриваємо файл Seating.as (можна любим текстовим редактором)
2) ставемо змінній webApplication значення false(тобто "public static var webApplication:Boolean = false")
3) компілюємо флешку
===================================================================================================
Веб версій:
1) відкриваємо файл Seating.as (можна любим текстовим редактором)
2) ставемо змінній webApplication значення true(тобто "public static var webApplication:Boolean = true")
3) ставемо змінній jsLoadFunction імя javascript метода, який буде викликатися зразу після ініціалізації флешки.
4) записуємо в массив domains всі доменні імя сайтів, на яких флешка буде знаходитися.(наприклад  "public static var domains:Array = ["localhost"]")
5) ставемо змінній developeMode значення false(тобто "public static var developeMode:Boolean = false")
6) компілюємо флешку

Флеш методи які доступні для JavaScript-а:
load(plannerFile:String) - метод для підгрузки .planner файла у флешку. У параметрі передається .planner файл у вигляді строки(String)
save() - метод повертає файл, який був змінений флешкою
resize(width:Number,height:Number) - метод для змінення розмірів флешки.
===================================================================================================
Компіляція:
1) скачуємо Adobe Flash CS4(скачати можна на офіційному сайті). 
2) установлюємо Adobe Flash CS4
3) відкриваємо файл project-editor.fla
4) нажимаємо File->Export->Export Movie
5) вказуємо путь, та імя файла, тип файла не змінюємо (залишаємо *.swf)
6) нажимаємо на кнопку "сохранить". після цього буде зкомпільований файл.
